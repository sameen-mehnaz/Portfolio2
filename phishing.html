PHISHING DETECTION

The phishing detection system leverages machine learning and deep learning models to identify and prevent phishing attacks. It uses technologies like Python, FastAPI, and MySQL, along with models such as Logistic Regression, Random Forest, Naive Bayes, SVM, CNNs, and RNNs. Feature engineering techniques like TF-IDF and Bag-of-Words enhance detection accuracy by analyzing website URLs and attributes. This solution strengthens cybersecurity by safeguarding users against fraudulent websites and potential data breaches.
